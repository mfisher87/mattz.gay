## Development

Use `python preview.py` or any other HTTP server to preview the page.


### Redirection

Redirection is annoying during development. So, when visiting this page deployed at
`localhost`, redirection is automatically disabled.

You can override the automatic behavior with the URL query parameter `?redirect` or
`?noredirect`.
